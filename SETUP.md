# Настройка фронтенда

## Установка зависимостей

```bash
cd front
npm install
```

Это установит `socket.io-client` и другие зависимости.

## Настройка переменных окружения

Создайте файл `.env` в папке `front` (опционально):

```env
VITE_API_URL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000
```

По умолчанию используются эти значения, если переменные не заданы.

## Запуск

```bash
npm run dev
```

## Использование

1. Откройте главную страницу
2. Вставьте ссылку на YouTube видео
3. Нажмите "Create room"
4. Поделитесь ссылкой на комнату с другими пользователями
5. Видео будет синхронизироваться в реальном времени для всех участников

## Функции

- ✅ Создание комнаты с YouTube видео
- ✅ Синхронизация воспроизведения (play/pause)
- ✅ Синхронизация перемотки
- ✅ Отображение количества участников
- ✅ Копирование ссылки на комнату
